# genAI-learning

A personal learning repository for experiments, notes, and small projects related to generative AI.

## Overview

This repository collects notes, examples, and small experiments while learning about generative models, prompt engineering, and related tools.

## Contents

- `notebooks/` - exploratory Jupyter notebooks
- `examples/` - small runnable examples and demos
- `notes/` - markdown notes and references

## Getting Started

1. Clone the repository:

   git clone https://github.com/<your-username>/genAI-learning.git

2. Create a virtual environment and install dependencies (if applicable):

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

3. Open the project in VS Code and explore the `notebooks/` and `examples/` folders.
